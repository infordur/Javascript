Array

Extender el objeto Array para que permita añadir nuevos elementos al final del array:

var array1 = [0, 1, 2];
array1.anadir(3);
// array1 = [0, 1, 2, 3]


Incluir la opción de controlar si se permiten elementos duplicados o no:
var array1 = [0, 1, 2];

array1.anadir(2);
// array1 = [0, 1, 2, 2]

array1.anadir(2, false);
// array1 = [0, 1, 2]
