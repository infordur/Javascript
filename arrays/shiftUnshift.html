Shift/Unshift

Write a code to alert a random value from array arr:
var arr = ["Plum","Orange","Donkey","Carrot","JavaScript"]
P.S. The code to get a random integer from min to max (inclusive) is:
var rand = min + Math.floor(Math.random()*(max+1-min))

    2. Los métodos shift y unshift sirven para eliminar o añadir elementos al principio del array.

    4. Si, en shift devuelve el tamaño final del array, y en el caso de unshift también devuelve el tamaño del array
