Acordeón

Descripción

El acordeón de JQuery UI consiste en un elemento que se expande o contrae mostrando u ocultando información.

Código

HTML

			<div id="accordion">
			  <h3 Descripción</h3>
			  <div>
			    <p>El acordeón de JQuery UI consiste en un elemento que se expande o contrae mostrando u ocultando información.</p>
			  </div>
			  <h3>Código</h3>
			  <div>
			    
			  </div>
			  <h3>Section 3</h3>
			  <div>
			      <a href="http://jqueryui.com/accordion/#collapsible">Enlace a la página del sitio oficial</a>
			  </div>
			</div>

JQuery

		$(document).ready(function() {
			$( "#accordion" ).accordion({
		       heightStyle: "content"
			});
		});

Descripción

El autocomplete de JQuery UI es el típico cuadro que sugiere opciones a la hora de ir escribiendo.

Código

HTML

	   		<label for="search">Buscar: </label>
			<input id="search">

JQuery

		$(document).ready(function() {
			$.widget( "custom.catcomplete", $.ui.autocomplete, {
			    _create: function() {
			      this._super();
			      this.widget().menu( "option", "items", "> :not(.ui-autocomplete-category)" );
			    },
			    _renderMenu: function( ul, items ) {
			      var that = this,
			        currentCategory = "";
			      $.each( items, function( index, item ) {
			        var li;
			        if ( item.category != currentCategory ) {
			          ul.append( "<li class='ui-autocomplete-category'>" + item.category + "</li>" );
			          currentCategory = item.category;
			        }
			        li = that._renderItemData( ul, item );
			        if ( item.category ) {
			          li.attr( "aria-label", item.category + " : " + item.label );
			        }
			      });
			    }
			});

		    var data = [
		      { label: "Chorizo", category: "Carnes" },
		      { label: "Salchichón", category: "Carnes" },
		      { label: "Jamon York", category: "Carnes" },
		      { label: "Plátano", category: "Frutas" },
		      { label: "Pomelo", category: "Frutas" },
		      { label: "Kiwi", category: "Frutas" },
		      { label: "Doritos", category: "Snacks" },
		      { label: "Pelotazos", category: "Snacks" },
		      { label: "Patatas Fritas", category: "Snacks" }
		    ];
		 
		    $( "#search" ).catcomplete({
		      delay: 0,
		      source: data
		    });
		});

Descripción

El elemento slide de JQuery UI consiste en una barra que puedes desplazar para cambiar su valor.

Código

HTML

  	<label for="amount">Price range:</label>
 	<input type="text" id="amount" readonly style="border:0; color:#f6931f; font-weight:bold;">
	<div id="slider-range"></div>

JQuery

	$(document).ready(function() {
		$( "#slider-range" ).slider({
	      range: true,
	      min: 0,
	      max: 1000,
	      values: [ 75, 300 ],
	      slide: function( event, ui ) {
	        $( "#amount" ).val( "" + ui.values[ 0 ] + "€ - " + ui.values[ 1 ]+"€" );
	      }
	    });
	    $( "#amount" ).val( "" + $( "#slider-range" ).slider( "values", 0 ) +
	      "€ - " + $( "#slider-range" ).slider( "values", 1 )+"€" );
	});
